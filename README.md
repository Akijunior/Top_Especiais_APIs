# Top_Especiais_APIs

Avaliar APIs e a forma como utiliza-las em projetos.

Firebase: Ferramenta que possibilita armazenamento de dados no padrão nosql. Para tanto, gera-se uma chave do google e com ela é possível ter acesso a API.

CatAPI: Ferramenta que possibilita obtenção de imagens de gatos online por meio de uma chave de API.

Oxford Dictionaries: Ferramenta que oferece dados acerca de uma dada palavra pesquisa, como sinônimos e traduções, por meio de uma chave de api gerada pós registro no site oficial.
